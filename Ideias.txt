    - O Fluxo Técnico (Passo a Passo)
        Independentemente da opção, o fluxo no seu código Python será este:
        - Recebimento: O usuário envia a imagem via formulário (Multipart/Form-Data) para sua API.
        - Renomeação (Segurança): Nunca salve com o nome original (ex: "foto.jpg"). Gere um nome único (UUID) para evitar que uma imagem sobrescreva outra.
        - Upload: Seu código Python envia o arquivo para o Storage (Nuvem ou Pasta).
        - Obtenção da URL: O Storage confirma que salvou e te dá o caminho.
        - Persistência: Você pega esse caminho (URL) e faz o INSERT no seu banco de dados na coluna VARCHAR(2048).
